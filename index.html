Evolution of the World Wide Web

The World Wide Web (WWW), created by Tim Berners-Lee in 1989, started as a tool for sharing information among scientists. Today, it is an essential part of daily life.

Key Terms

  • HTML – Markup language used to structure web content
  • HTTP – Protocol for transmitting web data
  • Web Browser – Application used to access websites
