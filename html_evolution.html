HTML Evolution

HTML (Hypertext Markup Language) enabled the creation of linked documents on the web. Early versions (1.0, 2.0) were limited. HTML4 (1997) added multimedia and scripting support.

VersionYearFeatures
HTML 1.01993Basic text and links
HTML 4.01997CSS and multimedia
HTML52014Video, audio, canvas
